Honeybee VTK Command Line Interface¶

honeybee_vtk¶

Honeybee VTK commands entry point.

honeybee_vtk [OPTIONS] COMMAND [ARGS]...

translate¶

Translate a HBJSON file to several VTK, XML, or HTML file.

The output file is either a zipped file that contains all the generated VTK/XML files or an HTML file.

Args:
hbjson-file: Path to input HBJSON file.
honeybee_vtk translate [OPTIONS] HBJSON_FILE

Options

-n, --name <name>¶

Name of the output file. If not provided, the name of input HBJSON file will be used.

-f, --folder <folder>¶

Path to target folder.

Default

.

-t, --file-type <file_type>¶

Switch between VTK, XML, and HTML formats. Default is HTML.

Default

html

Options

vtk | xml | html

-ig, --include-grids¶

Export grids.

Default

False

-is, --include-sensors <include_sensors>¶

Export sensors as either arrows or color-grouped points.

Options

vectors | points

-in, --include-normals <include_normals>¶

Export aperture normals as either arrows or color-grouped points.

Options

vectors | points

-sh, --show-html, --show¶

Open the generated HTML file in a browser.

Default

False

Arguments

HBJSON_FILE¶

Required argument
